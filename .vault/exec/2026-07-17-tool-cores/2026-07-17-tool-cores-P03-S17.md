## Scope

## Description

- Add `test_document_agent_invokes_rag_search_midturn_and_cites` as a sibling case of the S05 floor harness in `test_tool_cores_floor_live.py` (extend, do not fork): reuse `AcceptanceHarness` run-start, the SSE observation, the document-dir write-watcher, and the `S05_PROFILE` override.

An earlier raw-frame probe (`pw7-1784281614`) captured the same behaviour and fixed the observation surface.

The tool-call did NOT surface as a `tool_call_start` SSE frame (zero tool-call frames, same as S05) - it surfaces in the agent's `message_chunk` narration, so the assertions key on the message stream plus citation resolution, with the daemon `/search` log recorded here as the corroborating trace.
