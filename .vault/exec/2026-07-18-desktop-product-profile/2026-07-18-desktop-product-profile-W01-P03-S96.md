- Exercise both direct public APIs through live caller-owned generation leases.

deterministic ZIP composition inside unpublished generations. The matrix proves exact
create-new winners, deterministic bytes and evidence, continuous authority under
substitution, fail-closed late errors with retained poison, and no redirected outside
bytes. Both the 80,001-member archive boundary and 80,001-entry source-tree boundary
are exercised through the public APIs using production-owned limits.
