Edge conformance mapping
========================

The maintained edge mapping records how external verbs reach the runtime.

:download:`Open the edge conformance verb mapping
<a2a-edge-conformance-verb-mapping.md>`.

Use the mapping with :mod:`vaultspec_a2a.api`,
:mod:`vaultspec_a2a.protocols`, :mod:`vaultspec_a2a.control`, and
:mod:`vaultspec_a2a.streaming`.

The Markdown mapping remains the source document. This wrapper gives it a
stable Sphinx navigation target.
