# Wire Types Gap Analysis

## 1. Codegen Status

`wire-types.ts` was generated by `openapi-typescript` from FastAPI's
`/openapi.json` endpoint. The file header says "auto-generated by
openapi-typescript." No `generate-types` script exists in `package.json`.
The regeneration command (per ADR-011 audit notes) is:

```bash
npx openapi-typescript http://localhost:8000/openapi.json -o src/app/data/wire-types.ts
```

This requires the API server to be running. No CI step validates freshness.

**Recommendation**: Add `"generate:types": "openapi-typescript http://localhost:8000/openapi.json -o src/app/data/wire-types.ts"` to `package.json` scripts. Regenerate after each backend schema change.

## 3. Regeneration Strategy

Since the file is auto-generated, the cleanest fix is:

3. The 4 missing fields will appear automatically
