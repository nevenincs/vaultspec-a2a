# A2A Protocol Technical Specifications (v1)

 the normative A2A protocol definitions based on the
`proto3` specification.

## 1. Core Service Operations (A2AService)

| Method | Request Type | Response Type | Description |
| :--- | :--- | :--- | :--- |
| `SendMessage` | `SendMessageRequest` | `SendMessageResponse` | Sends a message to the agent. |
| `SendStreamingMessage` | `SendMessageRequest` | `stream StreamResponse` | Streaming version of SendMessage. |
| `GetTask` | `GetTaskRequest` | `Task` | Retrieves current state of a task. |
| `ListTasks` | `ListTasksRequest` | `ListTasksResponse` | Lists tasks with filtering/pagination. |
| `CancelTask` | `CancelTaskRequest` | `Task` | Cancels an active task. |
| `SubscribeToTask` | `SubscribeToTaskRequest` | `stream StreamResponse` | Subscribes to real-time task updates. |
| `GetExtendedAgentCard` | `GetExtendedAgentCardRequest` | `AgentCard` | Returns detailed card for authenticated agents. |

## 2. Core Data Structures

### Task

The core unit of action in A2A.

- `id` (string, REQUIRED): Unique UUID generated by the server.
- `context_id` (string, REQUIRED): UUID for the collection of interactions.
- `status` (TaskStatus, REQUIRED): Current state and message.
- `artifacts` (repeated Artifact): Output results.
- `history` (repeated Message): Interaction history.
- `metadata` (google.protobuf.Struct): Custom key/value storage.

### Message

A single unit of communication.

- `message_id` (string, REQUIRED): Unique UUID created by the sender.
- `context_id` (string): Associated context.
- `task_id` (string): Associated task.
- `role` (Role, REQUIRED): `ROLE_USER` (1) or `ROLE_AGENT` (2).
- `parts` (repeated Part, REQUIRED): Content containers.
- `metadata` (google.protobuf.Struct): Optional metadata.
- `extensions` (repeated string): URIs of active extensions.
- `reference_task_ids` (repeated string): References to other tasks.

### Part

A container for a section of communication content.

- `content` (oneof):
  - `text` (string): Textual content.
  - `raw` (bytes): File content (base64 in JSON).
  - `url` (string): URL to content.
  - `data` (google.protobuf.Value): Structured JSON data.
- `metadata` (google.protobuf.Struct): Optional metadata.
- `filename` (string): Optional filename.
- `media_type` (string): MIME type (e.g., `application/json`).

### Artifact

Task outputs.

- `artifact_id` (string, REQUIRED): Unique within the task.
- `name` (string): Human-readable name.
- `description` (string): Optional description.
- `parts` (repeated Part, REQUIRED): Content (min 1 part).
- `metadata` (google.protobuf.Struct): Optional metadata.
- `extensions` (repeated string): Extension URIs.

## 3. Enums and Statuses

### TaskState

- `TASK_STATE_UNSPECIFIED` (0)
- `TASK_STATE_SUBMITTED` (1): Created/Acknowledged.
- `TASK_STATE_WORKING` (2): Actively processing.
- `TASK_STATE_COMPLETED` (3): Finished (Terminal).
- `TASK_STATE_FAILED` (4): Failed (Terminal).
- `TASK_STATE_CANCELED` (5): Canceled (Terminal).
- `TASK_STATE_INPUT_REQUIRED` (6): Interrupted (Waiting for user).
- `TASK_STATE_REJECTED` (7): Agent refused (Terminal).
- `TASK_STATE_AUTH_REQUIRED` (8): Interrupted (Waiting for auth).

### TaskStatus

- `state` (TaskState, REQUIRED)
- `message` (Message): Associated message.
- `timestamp` (google.protobuf.Timestamp): ISO 8601 recording time.

## 4. Agent Discovery (AgentCard)

### AgentCard

- `name` (string, REQUIRED)
- `description` (string, REQUIRED)
- `supported_interfaces` (repeated AgentInterface, REQUIRED)
- `provider` (AgentProvider)
- `version` (string, REQUIRED)
- `documentation_url` (optional string)
- `capabilities` (AgentCapabilities, REQUIRED)
- `security_schemes` (map<string, SecurityScheme>)
- `security_requirements` (repeated SecurityRequirement)
- `default_input_modes` (repeated string, REQUIRED)
- `default_output_modes` (repeated string, REQUIRED)
- `skills` (repeated AgentSkill, REQUIRED)
- `signatures` (repeated AgentCardSignature)
- `icon_url` (optional string)

### AgentInterface

- `url` (string, REQUIRED): Absolute HTTPS URL.
- `protocol_binding` (string, REQUIRED): `JSONRPC`, `GRPC`, or `HTTP+JSON`.
- `tenant` (string): Optional tenant ID.
- `protocol_version` (string, REQUIRED): e.g., "1.0".

## 5. Streaming and Events

### StreamResponse

A wrapper for streaming data containing a `oneof payload`:

- `task` (Task)
- `message` (Message)
- `status_update` (TaskStatusUpdateEvent)
- `artifact_update` (TaskArtifactUpdateEvent)

### TaskArtifactUpdateEvent

- `task_id` (string, REQUIRED)
- `context_id` (string, REQUIRED)
- `artifact` (Artifact, REQUIRED)
- `append` (bool): If true, append to previous artifact with same ID.
- `last_chunk` (bool): If true, final chunk of the artifact.
