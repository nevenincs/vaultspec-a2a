# A2A Walkthrough Reference

- **Source:** [https://github.com/holtskinner/A2AWalkthrough](https://github.com/holtskinner/A2AWalkthrough)
- **Relevance:** 8/10 - Hands-on educational material for A2A and MCP.

## Overview

This repository contains a series of Jupyter notebooks and Python scripts that
provide a step-by-step walkthrough of implementing and using the Agent-to-Agent
(A2A) protocol, often in conjunction with the Model Context Protocol (MCP).

## Key Content

- **Notebooks (1-8):** Cover topics from basic QA to complex scenarios like:
  - `2_A2AServer-PolicyAgent.ipynb`: Hosting an A2A Server.
  - `3_A2AClient.ipynb`: Using an A2A Client.
  - `6_A2AxMCPLangGraph.ipynb`: Integration of A2A, MCP, and LangGraph.
- **Example Agents:**
  - `a2a_healthcare_agent.py`: A domain-specific A2A agent.
  - `a2a_policy_agent.py`: Agent for policy-related tasks.
  - `a2a_research_agent.py`: Agent for research tasks.
- **MCP Integration:**
  - `mcpserver.py`: Example MCP server implementation.

## Educational Value

The walkthrough is particularly valuable for understanding how A2A agents can
be orchestrated to perform sequential tasks and how they can leverage local
tools via MCP.
